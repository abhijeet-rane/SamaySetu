# Contributing

> [!IMPORTANT]
> **This is a proprietary project with pending patent applications.**
>
> SamaySetu is not an open-source project. The production source code is maintained in a private repository, and this public repository serves as an architectural showcase only.

---

## Reporting Issues

If you find issues with the documentation in this showcase repository, you are welcome to:

1. **Open an Issue** — Describe the problem with the documentation, broken links, or factual errors.
2. **Suggest Improvements** — If you believe the documentation could better explain a concept, open an issue with your suggestion.

---

## What This Repository Does NOT Accept

- ❌ Pull requests with code changes
- ❌ Feature requests for the product
- ❌ Bug reports for the live application
- ❌ Requests to access the private source code

---

## Code of Conduct

All interactions in this repository must be professional and respectful. Violations of this expectation may result in being blocked from the repository.

---

## Contact

For licensing inquiries, partnership discussions, or academic collaboration proposals, please open an issue with the `[inquiry]` tag.
